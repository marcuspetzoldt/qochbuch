== README

=== qochbuch, a shared collection of recipes.

* Ruby 2.0.0
* Rails 4
* PostgresSql

=== *qochbuch* features

* anonymous users
  * browse recipes
  * find recipes by tag
  * find recipes by full text search
  * change servings

* additional features for signed in users
  * create, edit and tag recipes
  * upload pictures
  * rate recipes of other users
